# Secretaria Digital SENAI

Uma reformulação moderna da plataforma da Secretaria Digital do SENAI, desenvolvida com foco em melhorar a experiência do usuário, modernizar a interface e tornar a navegação mais intuitiva e acessível.

---

## Sobre o Projeto

Este projeto consiste na recriação visual da Secretaria Digital do SENAI, trazendo uma nova identidade visual, melhorias na organização das informações e uma interface mais limpa e responsiva.

O projeto surgiu a partir da percepção de que o site atual apresenta um visual ultrapassado, problemas de usabilidade e alguns recursos pouco funcionais. A proposta foi criar uma versão mais moderna, organizada e visualmente agradável, mantendo a essência da plataforma original.

O objetivo principal foi praticar desenvolvimento front-end, estruturação de páginas, design responsivo e construção de interfaces modernas inspiradas em sistemas atuais.

---

## Funcionalidades

- Página inicial moderna
- Interface reformulada
- Navegação intuitiva
- Área de login estilizada
- Organização visual aprimorada
- Layout responsivo
- Melhor experiência de usuário
- Estrutura inspirada em plataformas modernas

---

## Tecnologias Utilizadas

- HTML5
- CSS
---

## Objetivos do Projeto

- Modernizar visualmente a Secretaria Digital do SENAI
- Melhorar a usabilidade da plataforma
- Praticar desenvolvimento front-end
- Trabalhar responsividade e organização de layout
- Criar uma interface mais agradável e funcional
